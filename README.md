# work.studio
A modern, web interface built using HTML, CSS, and JavaScript, featuring smooth animations with GSAP and Locomotive Scroll. The project includes interactive hover effects, dynamic layouts, and responsive design elements to create a visually engaging user experience. It focuses on creative UI/UX, scroll-based animations, structured component design.
